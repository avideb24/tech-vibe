###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A:{}

<i>Explanation: Here greeting and greetign are behave like two different variables because JavaScript is case sensitive. greeting variable is declared globally but not assigned the value, so it will return undifined. But greetign has assign a value "{}". So it's return {}.</i>

</p>
</details>
